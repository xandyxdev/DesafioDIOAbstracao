📚 Desafio DIO - Abstração em Java
📌 Sobre

Projeto em Java simulando um Bootcamp, onde desenvolvedores se inscrevem em cursos e mentorias, progridem e acumulam XP.

🧠 Conceitos usados
POO (Abstração, Herança, Encapsulamento)
Set / LinkedHashSet
Stream API (map, sum)
Optional
⚙️ Funcionalidades
Inscrever em bootcamp
Progredir nos conteúdos
Separar conteúdos inscritos e concluídos
Calcular XP total
